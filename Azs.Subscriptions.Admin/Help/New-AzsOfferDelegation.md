# New-AzsOfferDelegation

## SYNOPSIS
Create a new offer delegation.

## SYNTAX

```
New-AzsOfferDelegation -Name <String> -OfferName <String> -SubscriptionId <String> -ResourceGroupName <String>
 [-Location <String>] [<CommonParameters>]
```

## DESCRIPTION
Create a new offer delegation.

## EXAMPLES
